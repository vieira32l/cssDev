CSS Seletores

Seletores CSS são utilizados para achar ou selecionar elementos HTML que você queira estilizar.

Podemos dividir seletores CSS em 5 categorias:

  • Seletores Simples: Seleciona o elemento baseado num nome, id ou classe.
  • Seletores Combinados: Seleciona elementos baseados em uma relação específica entre eles.
  • Seletores de Pseudo-Classe: Seleciona elementos baseados em um certo estado.
  • Seletores de Pseudo-Elemento: Seleciona e estiliza parte de um elemento.
  • Seletores de Atributo: Seleciona elementos baseados em um atributo ou atributo de valor.

O seletor CSS por ID

O seletor por id usa o atributo HTML id dentro de um elemento para identificar um elemento específico.
O id de um elemento é único em uma página, por isso ele deve ser usado para identificar um único elemento apena!
Para selecionar um elemento utilizando seu id específico usamos o hash(jogo da velha, #) seguido pelo nome do id.

Nota: O id não pode ser nomeado começando por um número.

O seletor CSS por Classe

O seletor por CSS por classe seleciona um ou vários elementos HTML que estão utilizando o atributo class.
Para selecionar elementos com uma classe específica, escreva um ponto (.) seguido pelo nome da classe.

O seletor CSS Universal

O seletor CSS universal utiliza o caractere(*) para selecionar todos os elementos de uma página.

Toda essa página está com todos os elementos estilizados universalmente com uma borda.

O seletor CSS agrupado

O seletor CSS agrupado seleciona toos os elementos HTML como as mesmas definições de estilo.

Por exemplo, caso você tenha 3 tags HTML estilizadas exatamente da mesma maneira seria melhor definir esse estilo uma única vez.
