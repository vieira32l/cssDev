CSS Sintaxe:

A sintaxe do CSS basicamente consiste na seguinte estrutura: SELETOR, DECLARAÇÃO, PROPRIEDADE e VALOR. O seletor é o elemento que vamos estilizar, a declaração é um bloco que contém uma ou mais declarações separadas por ponto e vírgula, cada declaração inclui uma propriedade CSS com seu valor separados por dois pontos.

Clique em F12 para ver o código fonte da página com os estilos css dessa página.
